# LaraLara
라라의 연무장 연구일지

메이플스토리 라라의 연무장 딜사이클을 평가하는 프로그램입니다.

## 파일 구성
 * **api_key.txt**: NEXON Open API 연동에 사용할 api key를 입력하는 파일입니다.
 * **nickname.txt**: 리플레이를 분석할 캐릭터의 닉네임을 한 줄에 하나씩 입력하는 파일입니다.
 * **main.py**: 정해진 기준에 따라 각 리플레이를 평가합니다.

## 산출물
 * **res.csv**: 닉네임, OCID, 리플레이 ID, 리스트레인트 링의 레벨, 모자의 재사용 대기시간 감소 효과, 측정 시간, 점수, 유효 여부를 출력한 파일입니다.
 * **detail.csv**: 점수를 각 스킬별로 분리하여 표기한 파일입니다.
 * **res 폴더**에는 각 리플레이마다 초당 점수 획득량을 시간대별로 표시한 그래프가 출력됩니다.
   초록색 부분은 스킬 조정에 의해 증가한 수치를 표시한 것입니다. 단, 기준 스탯 변동에 따른 점수 변화는 반영하지 않습니다.

## 설명 문서
 * **stat.md**: 기준 능력치를 계산하는 방법과 역대 기준 능력치를 계산한 문서입니다.
 * **skill.md**: 라라의 스킬을 설명하는 문서입니다.
 * **eval.md**: 점수를 계산하는 방식을 설명하는 문서입니다.
 * **update.md**: 스킬 변동 내역을 설명하는 문서입니다.
